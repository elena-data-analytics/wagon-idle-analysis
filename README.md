
# Аналитика критического простоя вагонов на станции дислокации

## Описание проекта
Проект посвящён анализу операционного среза данных вагонного парка с целью выявления факторов, ассоциированных с критическим простоем вагонов на станции дислокации. Анализ направлен на поиск зон операционной неэффективности без попыток экономической оценки.

## Бизнес-вопрос
При каких условиях и в каких сегментах вагонного парка возникает критический простой вагонов на станции дислокации?

## Данные
- Тип данных: операционный срез состояния вагонов  
- Единица анализа: один вагон  
- Объём выборки: 425 вагонов  
- Источник: внутренние операционные данные  

## Метрика и критерий
- Целевая метрика: простой на станции дислокации (дни)  
- Критический простой: более 5 суток  

## Подход к анализу
- предварительная проверка и валидация данных (EDA)  
- формирование признака критического простоя  
- описательный факторный анализ  
- сравнение долей критического простоя по управляемым сегментам  
- визуальный анализ с контролем размера выборки  

## Проверенные гипотезы
1. Доля критического простоя различается между дорогами дислокации  
2. Статус вагона связан с риском критического простоя  
3. Наличие текущей неисправности влияет на простой  

## Основные результаты
- Критический простой концентрируется на отдельных дорогах дислокации  
- Наибольший риск простоя наблюдается у вагонов под погрузкой и находящихся в ремонте  
- Фактор текущей неисправности не показал значимой связи с простоем в рамках имеющихся данных из-за малой доли наблюдений  

## Ограничения анализа
- данные представляют собой разовый срез без временной динамики  
- причинно-следственные выводы не делаются  
- финансовые эффекты и оборот не оцениваются  
- часть факторов представлена ограниченным числом наблюдений  

## Инструменты
Python, pandas, matplotlib
